어쨋든.. 나는 주? 개발은 웹 poc 개발하는 걸로 정해졌다. 앞으로 더 책임감을 가지고 해야겠다는 생각이 들었따. 어쨌든 문서상? 우선순위 상으로 web poc 개발을 하겠지만, 다같이 하게되서 그래도 다행이다. 누구하나 재미없고 하기 싫은 것을 혼자 도맡아 정말 너무나 싫다. 다같이 재밌고, 하기싫어도 다같이 하기 싫은게 났지..
